How to Add a New Procedure

Two things to do: (1) place your HTML file in the same folder, (2) add one entry to the procedures array below. That's it.

Step 1 — Create the procedure HTML file

Use the specification tab in any existing procedure (e.g. AOS) or provide the prompt template and your source .docx to Claude. Save the output as a standalone .html file.

The file must be in the same folder as this library page, or update the file path accordingly.

Step 2 — Add an entry to the procedures array

Open this file in any text editor (VS Code, Notepad++, even Notepad). Search for:

// ===== PROCEDURE REGISTRY =====

You'll see the array that drives the search dropdown. Add a new object at the end:

// ===== PROCEDURE REGISTRY =====
```

// To add a procedure: copy one line, change the 4 values.
const procedures = [
{
name: ‘Absentee Owner Surcharge (AOS)’,
category: ‘Land Tax’,
docRef: ‘DOC/18/509543 • Revised: 13 June 2025’,
file: ‘aos.html’
},
{
name: ‘TARB Referrals & Determinations for LRA’,
category: ‘TARB’,
docRef: ‘Revised: 26 November 2025’,
file: ‘tarbreferral.html’
},
{
name: ‘Build to Rent’,
category: ‘Concessions’,
docRef: ‘Created: 15 June 2023’,
file: ‘buildtorent.html’
},
{
name: ‘Insolvency’,
category: ‘insolvency’,
docRef: ‘Created: 15 June 2023’,
file: ‘insolvency.html’
},
name: ‘Property Clearance Certificate’,
category: 'Land Tax’,
docRef: ‘Created: 15 June 2023’,
file: ‘pcc.html’
},
];
```

The 4 values explained

name — The procedure title shown in search results. Use the full name.
category — A short grouping label (e.g. Land Tax, TARB, Exemptions). Used for search filtering.
docRef — The CM10 reference and/or revision date. Shown as metadata under the title.
file — The filename of the standalone HTML. Must match exactly including capitalisation.

Step 3 — Save and test

  1. Save this library HTML file.
  2. Open it in a browser.
  3. Search for your new procedure name.
  4. Click it — the procedure should load below the banner.

Troubleshooting

  • Procedure doesn't appear in search? Check for missing commas between entries in the array. Every entry except the last needs a comma after the closing }.
  • Blank frame after clicking? Check the filename matches exactly (case-sensitive). Check the file is in the same folder.
  • Mermaid diagrams don't render? The standalone file handles its own Mermaid — no action needed here.
