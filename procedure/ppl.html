Context: Primary Production Land Exemption Framework LEGISLATION & SECTIONS

Overview

This procedure identifies the process for applying a Primary Production Land (PPL) exemption for land used primarily for primary production under the Land Tax Act 2005.

```

Four Exemption Sections (s65-s68)

  • s65: Land OUTSIDE greater Melbourne used primarily for primary production
  • s66: Land IN greater Melbourne but NOT in urban zone, used primarily for primary production
  • s67: Land IN greater Melbourne IN urban zone, used solely/primarily for primary production (TARB decision)
  • s68: Land BEING PREPARED for primary production use (12-month max, renewable once)

Request Channels

  • My Land Tax (online portal)
  • Written correspondence (letter/email)
  • Phone call to SRO staff (AD/MLT processing)

Key Control Points

  • Investigation Flag: Red ! flag = do NOT process without investigator approval
  • Zoning Determination: Must check VicPlan to determine urban zone status
  • Section Selection: Location + zone determines section (s65/s66/s67)
  • TARB Referral: All s67 requests must go to TARB for decision
Authority: Land Tax Act 2005, Sections 65-70 | Time Standard: 15 minutes per PPL action item
Input data: PPL exemption request, customer details, property location/zoning
Processing responsibility: Staff (verification & application), e-Sys (exemption recording), TARB (s67 decisions)
Decision: Which section applies? Is exemption eligible? Proceed or escalate?
Output: PPL exemption applied or referred to TARB, customer notified, refunds processed if applicable
```

```
Investigation Flag Handling

If red ! flag OR "LTX Under Investigation" node exists: DO NOT PROCESS. Escalate to investigator immediately.

Previous Decision Handling

If exemption previously denied: Do not grant. Reply to customer advising of prior decision. Escalate if new information provided.

If exemption previously granted: Review to understand prior decision basis before processing amendment.

Investigation Flag Escalation Process

What is a red ! flag? Customer has active investigation OR completed investigation within last 3 years. Treat as high-risk.

What is "LTX Under Investigation" non-issue? Specific flag showing investigation number (IC-prefixed). Found in non-issue list on customer account.

Escalation Steps:

  1. Email investigator with: Customer ID, name, PPL request details, property
  2. Include action item reference number
  3. Ask investigator to advise if PPL processing should proceed or hold
  4. Do NOT process exemption until investigator approval received
  5. Document investigation flag and escalation in action item notes

Contact Information: Find investigator via e-Sys team directory or CMT (contact investigation team lead)

Previous Decision Handling

If Exemption Previously DENIED:

  • Check if denial was by investigation or TARB decision
  • Check if request is for SAME property or DIFFERENT property
  • Check if within investigation period (if investigation-based denial)
  • If same property & investigation-based denial: Refer back to investigator with new information (if provided)
  • Do NOT re-grant without investigator or TARB approval

If Exemption Previously GRANTED:

  • Review basis of prior grant (check comments for section and activity type)
  • If this is first-time request for property: Normal processing applies
  • If this is renewal/amendment: Check if changes affect eligibility (e.g., s68 2-year max)
```

```
Procedure Detail: VicPlan Step-by-Step, Zone Determination & Precinct Structure Plan (PSP) Check▼

Detailed VicPlan Navigation Process

Step 1: Access VicPlan

  1. Visit MapShareVic: maps.land.vic.gov.au
  2. Look for "VicPlan" option (should be prominent on main page)
  3. Click to launch VicPlan interface

Step 2: Search for Property

  1. Click Search box (usually top-left)
  2. Enter property address (street number, street name, suburb)
  3. OR enter Property ID/Parcel number if known
  4. Press Enter or click Search

Step 3: Identify Property on Map

  1. Map will pan to property location
  2. Property should be highlighted or marked
  3. Use zoom controls to get clear view of property boundaries

Step 4: Check Zone Layer

  1. On left panel, find "Layers" menu
  2. Look for "Zoning" or "Zones" layer
  3. Toggle ON the zoning layer (check mark)
  4. Map will now show zone colors/labels

Step 5: Determine Zone Type

  • Non-Urban Zone (Green, Gray, etc.): s66 eligible. Property may qualify for exemption if used for primary production.
  • Urban Zone (Red, Residential, Mixed Use, etc.): Could be s67 (refer to TARB) OR UGZ (check PSP)
  • Urban Growth Zone (UGZ): Special handling. See PSP check below.

Step 6: Save Zone Map

  1. Use browser Print function (Ctrl+P) or Screenshot tool
  2. Capture map with property clearly visible and zone highlighted
  3. Include date and address label on image
  4. Save to CM10 under customer record, amendments folder
  5. Note date saved in action item (e.g., "VicPlan map saved 1/10/2024")

Precinct Structure Plan (PSP) Check for UGZ Properties

What is a PSP? Planning document showing future development for Urban Growth Zone area. If PSP exists, property treated as s67 (refer to TARB).

How to Check for PSP:

  1. Refer to TARB guide on TaxNet: "How to find where land is located for PPL exemptions"
  2. Check Victoria Planning Portal: planning.vic.gov.au
  3. Search for property suburb/precinct
  4. If PSP found and active: Property is s67 (refer to TARB)
  5. If no PSP found: Property may be s66 (normal exemption processing)

Urban Growth Zone (UGZ) + NO PSP s66 Process exemption normally (if PPL eligible)
```

```
Grandfathering Clause - Complete Conditions

What is Grandfathering? Schedule 3(15) of Land Tax Act allows properties exempt under s65 or s68 prior to 2015 to continue that exemption despite boundary change.

Conditions for Grandfathering:

  • ☐ Property was exempt under s65 or s68 in 2014 or earlier
  • ☐ Property now falls within Greater Melbourne boundary (post 1 Jan 2015)
  • ☐ Property now falls in urban zone (would normally require s67)
  • ☐ Primary production activity commenced during customer's ownership before 1 Jan 2015
  • ☐ No sale, subdivision, or adjoining land acquisition has occurred since 2015

Grandfathering Termination Triggers (Exemption Ends):

  • ☐ Property is SOLD to new owner
  • ☐ Property is SUBDIVIDED (split into multiple titles)
  • ☐ Customer acquires ADJOINING LAND that is contiguous or separated only by road/railway

Customer Notification - Grandfathered Property Letter Template

"From 7 May 2014 the Government redefined the greater Melbourne boundary. This resulted in your property now falling within greater Melbourne and in an urban zone, which would mean more stringent primary production criteria under section 67 would apply.

However, under schedule 3(15) of the Act, as the primary production activity under your ownership commenced prior to this time, an exemption has been granted under section 65 of the Act.

The exemption will continue to be assessed under section 65 UNTIL:
• The land is disposed of
• The land is subdivided, OR
• You acquire other land contiguous with or separated only by a road/railway

If any of the above events occur, the exemption would no longer be assessed under section 65 and the more stringent criteria under section 67 would need to be met."
```

Q5 - Has the Primary Production Type Been Verified? IMPROVED

PPL Evidentiary Requirements (Must Verify All)

  • Start date of primary production activity on property
  • Type of primary production activity (cropping, livestock, cultivation, etc.)
  • If stock kept: stock type, number of animals held
  • Portion (%) of land used for PPL; use of non-PPL portion
  • Property classification code (from council rates notice)
  • Stock numbers are adequate for claimed activity type
  • Documentation/evidence supports claimed activity
```

Accepted Primary Production Types

  • Cultivation: Cropping (wheat, barley), orchards, vineyards, timber plantations (NOT mills), flowers for floristry
  • Animals/Poultry: Cattle, sheep, chickens, agisted stock (if offspring/produce sold)
  • Bees: Honey production
  • Commercial Fishing: Including preparation and storage
  • Plant Propagation: Nurseries (wholesale, NOT retail), mushrooms, orchids

NOT Eligible (Special Rules Apply)

  • Recreational horses (must be bred + offspring sold to qualify)
  • House & curtilage (code 111) unless on same title as farm AND not rented
  • Agistment of horses for recreational purposes
  • Secondary production (e.g., timber mill, not plantation)
Procedure Detail: Evidence Requirements Matrix, Stock Validation, House & Curtilage, Horse Breeding & 10% Land Use Rule▼

Evidence Requirements Matrix by Activity Type

Activity Required Evidence Stock Numbers
Cropping Crop type, planted area, harvest records N/A
Livestock Stock type, property location, breeding/sale records Minimum viable herd (see validation)
Orchards Orchard size, tree type, harvest records N/A (land-based)
Horse Breeding Breeding records, offspring sale records, NOT recreational Breeding stock + offspring
Bees Number of hives/honeycombs, honey production records Min. [need to verify from Revenue Ruling]
Commercial Fishing License, records of fish caught/sold, storage facilities N/A

Stock Number Validation Guidance

Cattle: Minimum 5-10 head considered viable breeding herd (verify against Revenue Ruling LTA-010)

Sheep: Minimum 20-30 animals considered viable flock

Horses: Must include breeding stock (stallion/mare) + offspring. Recreational horses NOT eligible.

Poultry: Minimum 100+ birds for commercial viability

Bees: Minimum 2-3 hives

Action if Stock Numbers Low: Request further information from customer. Can apply exemption if customer confirms activity ongoing, even if current count is lower.

House & Curtilage (Property Classification 111) - Edge Cases

General Rule: House & curtilage NOT eligible for PPL (treated as residential)

Exception: If house is on SAME TITLE as farm land, and house is occupied by the person conducting PPL activity, house can be included in exemption.

Examples:

  • ELIGIBLE: Farmer owns 100 hectares with farmhouse on same title, lives in farmhouse, conducts cropping
  • NOT ELIGIBLE: Same property but farmhouse is rented to tenant (use changes from PPL to rental)
  • NOT ELIGIBLE: House and farm on separate titles (contact council for clarification of classification)

Mornington Peninsula Example: Council may rate house and farm separately. If on same title, can include both in PPL exemption (if farm use primary).

Horsebreeding Documentation Requirements

Key Distinction: Horses kept for RECREATION do NOT qualify. Horses kept for BREEDING + SALE of offspring qualify.

Required Documentation:

  • Proof of breeding activity (mating records, stud book, etc.)
  • Proof of offspring sale (sale receipts, agistment contracts, stud records)
  • NOT just ownership of horses

Action if Proof Lacking: Request customer provide breeding records. Send template letter requesting horsebreeding documentation. Reserve for 21 days.

10% Non-PPL Land Rule (Partial Exemption Calculation)

Rule: If less than 10% of land is used for NON-primary production activities, apply full PPL exemption to entire property

Example: 100-hectare property: 95 hectares cropping, 5 hectares house + access = 5% non-PPL. Grant FULL exemption to all 100 hectares.

If 10% or More Non-PPL: Calculate exemption percentage. Example: 100 hectares, 15 hectares non-PPL = grant PPL on 85% of property.

Measurement Methodology:

  1. Identify total property size
  2. Calculate area used for primary production (cropping, livestock, etc.)
  3. Calculate area used for non-PPL (house, storage for non-PPL items, sheds for secondary production)
  4. If non-PPL ≥ 10%: Calculate exemption % = (PPL area / Total area) × 100%
  5. If non-PPL < 10%: Apply 100% exemption to entire property

Storage Classification: Storage FOR primary production (tractors, seed, fertilizer) = PPL. Storage for AFTER production (logs before milling) = non-PPL.

Source: Applying a PPL Exemption Procedure; Q5 Verification; Revenue Ruling LTA-010; Primary Production Evidence
Input data: Customer request, property details, supporting documentation
Processing responsibility: Staff (evidence review and validation)
Decision: PPL type verified? Evidence adequate? Stock numbers sufficient?
Output: PPL type confirmed or request further information; proceed to Q6
```

  1. Navigate to Property → Exemptions node

```
Section 68 Special Requirements (Expiry Date)

If applying s68 exemption:

  • Enter Expiry Date (12 months from start date)
  • Enter Termination Reason: "Ceased" or "Transitioned to primary production"
  • Document renewal or termination action in action item notes

  1. Recalculate customer liabilities for all affected years
  2. Issue amended assessments if overpayments occur
  3. Process refunds if applicable (refer to Refunds procedure)

Procedure Detail: e-Sys Field Entry, Section 68 Renewal Process, Refund Processing & 21-Day Further Information Hold▼

e-Sys Field Entry Guidance

Basis of Exemption Field: Select "LTX primary production land" (system recognizes PPL exemption type)

Start Date: Date primary production commenced (not application date). Can be retrospective if legitimate request.

Comment Field Examples (by section):

  • s65: "s65 PPL granted (cropping)" or "Outside metro boundary, s65 PPL (sheep)"
  • s66: "Casey Map 13/10/18 GWZ, s66 PPL (beef cattle)"
  • s68: "s68 preparation granted for 2023 only. Max 2 years."

Property Selection: Must tick checkbox next to property to apply exemption. If multiple properties on account, select only the qualifying property.

Exemption %: Usually 100%. If partial (non-PPL portion ≥ 10%), calculate % = (PPL area / total area) × 100

Section 68 Expiry Date: Enter date 12 months from start date (or 24 months if renewal granted). System will flag when expiry approaches.

Section 68 Renewal/Termination Process

Situation: Customer applied for s68 preparation exemption. Exemption expires after 12 months. What happens next?

Option 1: Primary Production Commenced (Convert to s65/s66)

  1. Customer contacts to advise primary production started
  2. Cancel s68 exemption (set termination date to date production commenced)
  3. Apply appropriate s65 or s66 exemption (based on location/zoning)
  4. Recalculate liability with new exemption

Option 2: Still Preparing (Renew s68 for Additional 12 Months)

  1. Customer requests additional 12-month preparation period (max 2 years total)
  2. Verify preparation activities continuing
  3. Extend expiry date by 12 months (new expiry = 24 months from original start)
  4. Document renewal in action item notes
  5. Notify customer of deadline for commencement of production

Option 3: Exemption Expires (Remove s68)

  1. If customer does NOT request renewal by expiry date
  2. Allow s68 exemption to expire naturally (system can auto-remove)
  3. Recalculate liability WITHOUT exemption for year following expiry
  4. Issue amended assessment showing full liability
  5. Note in action item: "s68 exemption expired [date]. No renewal request received."

Refund Processing Integration

If Exemption Creates Overpayment: Customer may have overpaid land tax in years when exemption should have been applied.

Refund Steps:

  1. Calculate overpaid amount per year
  2. Create refund request in e-Sys (Refunds module)
  3. Link refund to PPL action item (action item field)
  4. Select Root Cause: "EXEMPTION NOT FLAGGED (PPL)"
  5. Process refund via EFT (electronic funds transfer) to customer
  6. Refer to Land Tax Refunds procedure for detailed steps

Alternative to Refund: Allocate overpayment to future assessment liability (reduce next year's bill instead of refund)

Requesting Further Information - 21-Day Hold

Situation: Customer's request lacks required information (e.g., missing start date, unclear activity type, no stock numbers)

Process:

  1. Do NOT apply exemption yet
  2. Send email or letter requesting specific information (use PPL template)
  3. Include examples of acceptable documentation
  4. Request response within 14 days
  5. In e-Sys, reserve action item for 21 days in your worktray
  6. Set reminder/calendar alert for day 14 (follow-up if needed)

If Information Received: Process exemption using provided documentation

If Information NOT Received by Day 21: Close action item with comment "Further information requested [date]. No response received. Case closed." Unsolicited follow-ups by customer will reopen item.

Further Information Templates: Located in Templates OneNote file or LAT Correspondence Resource (DOC/18/642231). Includes specific templates for horsebreeding evidence requests.

Source: Applying a PPL Exemption Procedure; Q6 Apply Exemption; e-Sys Exemptions Node; Section 68 Renewal; Refunds Integration

Processing responsibility: e-Sys (exemption recording and liability recalculation)
Decision: Exemption % correct? Section 68 expiry needed? Refund required?
Output: Exemption applied, liabilities recalculated, assessments issued, customer notified
```

Procedure Summary

  • 6-step process (Q1-Q6) for processing PPL exemption requests
  • Q1: Pre-processing checks (investigations, prior decisions, allocation)
  • Q2: Location check (inside or outside Greater Melbourne boundary)
  • Q3: Zoning check via VicPlan (determines s66 vs s67)
  • Q4: Section selection matrix (location + zone = s65/s66/s67/s68) with grandfathering check
  • Q5: PPL type verification with evidence requirements matrix
  • Q6: e-Sys exemption application with liability recalculation and refund processing
  • Special handling for s67 (TARB referral), s68 (12-month max with renewal option), house & curtilage
  • Time standard: 15 minutes per PPL action item

Source of Authority

  • Division: Land Revenue Services
  • Branch: Land Revenue Administration
  • Created: 15 February 2024
  • Legislation: Land Tax Act 2005, Sections 65-70
  • Time Standard: 15 minutes per PPL action item

Definitions & Acronyms

PPL - Primary Production Land
s65 - Section 65 of Land Tax Act (land outside Greater Melbourne)
s66 - Section 66 of Land Tax Act (land in Greater Melbourne, non-urban zone)
s67 - Section 67 of Land Tax Act (land in Greater Melbourne, urban zone)
s68 - Section 68 of Land Tax Act (land being prepared for primary production)
Greater Melbourne - Area within Urban Growth Boundary (defined 1 Jan 2015)
UGZ - Urban Growth Zone
PSP - Precinct Structure Plan
TARB - Technical Advice and Review Branch (makes s67 decisions)
VicPlan - Online zoning mapping tool (MapShareVic)
CM10 - Content Manager (document storage system)
AD/MLT - Assisted Digital / My Land Tax (processing platform for s65 only)
Revenue Ruling LTA-010 - Primary production exemption guidance document
