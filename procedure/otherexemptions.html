What is an "Other Exemption"?NEW

✓ Understanding the exemption type is critical. "Other Exemption" is a distinct category that must be clearly differentiated from named exemptions.

```

Definition

"Other Exemption" means an exemption from land tax that does not fall within the standard named exemption categories (e.g., primary residence exemption, charitable exemption, religious exemption, etc.). It is applied under specific legislative authority when circumstances warrant exemption but don't fit standard categories.

Examples of "Other Exemptions" (QUALIFIES)

  • Crown land held for specific public purpose not covered by standard exemptions
  • Property used exclusively for emergency services depot or public infrastructure
  • Land held in trust for public benefit under specific legislative authority
  • Property exempted by specific Order-in-Council or ministerial decision
  • Temporary exemption for community benefit project (time-limited)

Examples of What is NOT "Other Exemption" (DOES NOT QUALIFY)

  • Primary residence exemption (use standard Primary Residence procedure)
  • Charitable exemption (use standard Charitable exemption procedure)
  • Religious exemption (use standard Religious exemption procedure)
  • Government owned land (use Crown property procedure)
  • Disability exemption (use standard Disability exemption procedure)

Key Distinction

Named Exemption: Property fits a pre-defined category with standard eligibility criteria. Example: "Residential property used as principal place of residence."

Other Exemption: Property requires individual assessment against specific legislative criteria. Example: "Crown land held for public benefit under Minister's direction."

THRESHOLD QUESTION: Does this property fit one of the named exemption types (Primary Residence, Charitable, Religious, Disability, etc.) OR does it require individual assessment as "other"?
Key point: If you're unsure whether this is "other exemption" vs. named exemption, proceed to Q0 below for explicit decision logic.
Input data: Property description, applicant claim, legislative context
Processing responsibility: Staff (initial categorization)
Decision: Is this "other exemption" requiring individual assessment? (YES→Q0 / NO→Route to named exemption procedure)
Output: Confirmed exemption type; proceed to appropriate procedure
```

Q0 - Is this claim an "other exemption" or a named exemption type?IMPROVED

🔀 ROUTING DECISION: This step determines which procedure applies. Route to wrong procedure = wrong outcome.

  1. Review Property & Claim: What is the property used for? What exemption is claimed?
  2. CHECK DECISION TREE:

    Does property fit one of these named categories?

    • IF Primary Residence: Owner-occupied residential property used as principal place of residence → Use PRIMARY RESIDENCE exemption procedure
    • IF Charitable: Property used exclusively for charitable purposes → Use CHARITABLE exemption procedure
    • IF Religious: Property used exclusively for religious purposes → Use RELIGIOUS exemption procedure
    • IF Disability: Property of person with disability used for disability support → Use DISABILITY exemption procedure
    • IF None of above: Property requires individual assessment → PROCEED TO Q1 (Other Exemption process)
  3. Document Routing Decision: Note which category applies or confirm "other exemption" path
  4. Decision: Is this "other exemption" requiring individual assessment?
CRITICAL: Routing to wrong procedure will cause application rejection and customer frustration. If uncertain, ask supervisor before proceeding.
Input data: Property type, claimed exemption category, applicant documentation
Processing responsibility: Staff (initial routing)
Decision: Named exemption type OR other exemption? (Route accordingly)
Output: Confirmed routing; proceed to named exemption procedure OR to Q1 for other exemption process

Q1 - Does the applicant and property meet basic eligibility criteria?IMPROVED

✓ Before investing time in document collection, verify basic eligibility. If ineligible, reject application immediately.

  • Property is within Victorian jurisdiction (SRO has authority)
  • Property is land subject to land tax (not exempt property type like Government-owned)
  • Applicant has standing to claim exemption (owner, registered proprietor, or authorized representative)
  • Application received in correct format (written claim with supporting documents)
  • No previous rejection for same claim without material change in circumstances
  • Property is not subject to active dispute or legal challenge
ELIGIBILITY DECISION: Do ALL six criteria above apply? YES→Proceed to Q2. NO→Reject application with clear explanation.
If rejecting, provide written notice explaining which eligibility criterion was not met and what would be needed to reapply.
Input data: Property record, applicant details, claim documentation
Processing responsibility: Staff (eligibility assessment)
Decision: All eligibility criteria met? (YES/NO)
Output: Proceed to Q2 if eligible; or reject with explanation if ineligible

Q2 - What type of application is this - new, renewal, amendment, or appeal?NEW

📋 APPLICATION PATH DECISION: Each path requires different documentation and process steps. Choose the correct path.

Which application type applies?

  • NEW APPLICATION: Property has never had "other exemption" before. Go to Q3 (collect full documentation)
  • RENEWAL: Property previously had exemption that expired or requires annual renewal. Go to Q3 (use renewal checklist in documentation section)
  • AMENDMENT: Property has exemption but circumstances changed (address, use, ownership). Go to Q3 (collect amendment documentation only)
  • APPEAL/CHALLENGE: Applicant disputes previous decision to grant or deny exemption. Reference APPEALS PROCESS section in Summary view; may require separate procedure
Why path matters: New applications require full validation. Renewals use abbreviated process. Amendments focus on changed circumstances only. Appeals require review authority, not initial assessment.
Input data: Current exemption status, application type indicator
Processing responsibility: Staff (path determination)
Decision: Application type - New/Renewal/Amendment/Appeal? (Route accordingly)
Output: Confirmed application path; proceed to Q3 (or appeal process if applicable)

Q4 - Make the exemption decision - approve or rejectIMPROVED

✓ Based on verified documentation and statutory criteria, determine if exemption should be granted or denied.

  1. Review Decision Criteria: The following questions must ALL be answered YES for approval:
    • Is there clear legislative basis for exemption (Act/Regulation/Order)?
    • Does applicant meet all conditions of that legislation?
    • Is property actually used for claimed exempt purpose?
    • Is exemption consistent with prior decisions on similar cases?
    • Is there no public policy reason to deny exemption?
  2. Decision Logic:

    ALL questions above = YES: APPROVE exemption. Issue notice of exemption grant.

    ONE or more = NO: REJECT exemption. Issue notice of rejection with reasons.

  3. Prepare Written Decision: Whether approving or rejecting, provide written notice including:
    • Decision (approved/rejected)
    • Effective date (if approved)
    • Reasons for decision (legal and factual basis)
    • Duration of exemption (if approved)
    • Renewal requirements (if approved)
    • Appeal/review process (if rejected)
  4. Escalation for Complex Cases: If ANY of the five criteria above is unclear or borderline, escalate to supervisor before finalizing decision (see Escalation Matrix)
  5. Decision: Has exemption decision been made and documented?
Do NOT approve exemption if all five criteria are not clearly met. When in doubt, escalate - don't guess.
Input data: Verified documentation, statutory criteria, prior decisions on similar cases
Processing responsibility: Staff (initial assessment), Supervisor/Manager (complex cases - escalation)
Decision: Exemption approve or reject based on five criteria? (APPROVE/REJECT)
Output: Written decision notice; proceed to Q5 if approved; or communicate rejection

Q5 - Apply exemption in TaxNet and communicate to customerIMPROVED

✓ After approval, apply exemption to system and notify customer of effective date and renewal requirements.

  1. Configure in TaxNet:
    • Access TaxNet property record
    • Navigate to Exemptions section
    • Add new exemption: Type = "Other Exemption", Category = [specific type], Effective Date = [approval date]
    • Set expiry date based on duration table (see Summary view)
    • Record renewal trigger date (typically 30 days before expiry)
    • Save and verify exemption applies to current assessment
  2. Calculate Impact: TaxNet will recalculate land tax excluding exempt land. Verify result is reasonable.
  3. Issue Updated Assessment: Generate and mail updated assessment showing exemption applied (if property was previously assessable)
  4. Communicate with Customer: Send written notice including:
    • Decision to grant exemption
    • Effective date (usually date of approval)
    • Duration of exemption (see Duration table)
    • Renewal requirements and timeline
    • Contact details for questions
  5. Schedule Renewal: Add to calendar: Renewal review to occur 30 days before exemption expiry date
  6. Record-Keeping: File decision notice, verification documents, and TaxNet configuration snapshot (see Record-Keeping Checklist in Summary)
  7. Decision: Has exemption been applied in TaxNet and customer notified?
CRITICAL: Exemption is not effective until BOTH (1) TaxNet configured AND (2) Customer notified. Don't do one without the other.
Input data: Approved exemption decision, property record, customer contact details
Processing responsibility: Staff (TaxNet configuration and customer communication), TaxNet system (assessment recalculation)
Decision: Exemption applied in TaxNet and customer notified? (YES/NO)
Output: Exemption active in system; customer informed; renewal scheduled; process complete

Eligibility Criteria - Q1 Detailed Explanation

These six criteria must ALL be satisfied. If even one fails, application must be rejected with written explanation.

```

1. Victorian Jurisdiction

Requirement: Property must be located within Victoria and subject to Victorian land tax law. SRO cannot grant exemptions for property outside jurisdiction.

How to Verify: Check property address against State boundaries. Verify property is Victorian land tax assessable property (not NSW, SA, etc.).

2. Land Subject to Tax

Requirement: Property must be land (not improvements) that is ordinarily subject to land tax. Some land types are automatically exempt by law (e.g., certain Crown land) and don't require "other exemption" application.

How to Verify: Check valuation record. Confirm property has been assessed for land tax in current or prior year (not automatically exempt category).

3. Applicant Standing

Requirement: Applicant must have legal standing - either registered proprietor (owner), or authorized representative with documented power of attorney.

How to Verify: Check Land Titles Office title register. Confirm applicant name matches registered proprietor OR review power of attorney document if represented.

4. Correct Application Format

Requirement: Application must be in writing with supporting documents. Verbal requests or incomplete applications don't proceed.

How to Verify: Review application submission. Ensure written claim + supporting docs provided. If incomplete, request missing items in writing.

5. No Prior Rejection Without Material Change

Requirement: If exemption was previously rejected, don't re-process identical claim. Only accept re-application if circumstances have materially changed.

How to Verify: Search exemption history for property. If previous rejection found, compare new application to prior rejection. If unchanged, reject immediately as frivolous.

6. No Active Dispute

Requirement: Property cannot be subject to active legal dispute, court challenge, or unresolved dispute notice.

How to Verify: Check SRO dispute register. Contact Land Revenue Branch for any known legal proceedings. If dispute found, hold application pending resolution.

```

Documentation Guide - Complete Reference

Use this guide alongside Q3 in the Staff view. This provides detailed guidance on each required document.

```

1. Property Identification Documents

  • Land Title: Obtain current title from Landata or Land Titles Office. Required fields: Volume/Folio, Crown Allotment, or Parish/Lot number.
  • Property Address: Full street address, suburb, postcode. Note: must match TaxNet assessment address.
  • Map Reference: If available, provide cadastral map coordinates or council map reference.

2. Ownership Verification

  • Title Register Extract: Current title page from Land Titles Office. Must show registered proprietor name matching applicant.
  • If Crown Land: Provide Crown Grant or Crown Land Lease document instead of private title.
  • If Trustees/Company: Provide corporate title and evidence of authority (board resolution, etc.).

3. Property Use Statement

  • Written Description: 1-2 page statement describing how property is currently used. Example: "Property is used exclusively as community center providing free services to local residents."
  • Supporting Lease/Agreement: If property is leased, provide lease showing intended use.
  • Management Plan: If required by exemption type, provide management/operating plan for the property.

4. Legislative Basis

  • Reference to Legislation: Applicant must cite specific Act, Regulation section, or Order-in-Council. Example: "Exemption claimed under Land Tax Act 2005 s[XX] for Crown land used for public benefit."
  • Verification: You must independently verify cited legislation actually allows such exemption. Check legislation library.

5. Supporting Evidence

  • Council Approval: If needed, provide council letter approving community use of property.
  • Insurance/License: If property is operated, provide public liability insurance policy or license.
  • Financial Evidence: For charitable use, provide audited accounts or charity registration.
  • Ministerial Authority: If exemption granted by order, provide copy of the ministerial decision or Order-in-Council.

6. Applicant Statutory Declaration

  • Format: Must be signed statutory declaration (not just letter). Signed before Justice of the Peace or lawyer.
  • Content: Declaration must state all facts in the application are true and correct to applicant's knowledge.
  • Date: Must be recent (within last 3 months). Older declarations may need to be updated.

7. Contact Details

  • Name, Address: Full contact details for applicant or representative.
  • Phone, Email: Best contact method. Ensure provided details are current and monitored.
  • Authorization: If representative applying, provide letter authorizing them to act.
```

Different "other exemption" types have different durations. Use this table to determine renewal timing.

```
Exemption Type Duration Renewal Trigger Notes
Crown Land (Public Purpose) Indefinite (until Crown status changes) Annual verification recommended Continues until property transferred out of Crown ownership
Emergency Services Indefinite (while in use) Review if use changes Continues while property used for emergency services
Community Benefit (Temporary) 2-5 years (as approved) 30 days before expiry date Requires renewal application to extend
Order-in-Council As specified in Order As specified in Order Check Order document for specific duration
Ministerial Direction As specified in direction As specified in direction Review direction document for renewal requirements
Transitional Exemption 1 year (renewable) 30 days before expiry Designed as temporary; requires annual renewal application

Renewal Process

  1. Trigger: 30 days before exemption expiry date, send renewal notice to property owner
  2. Applicant Response: Owner must submit renewal application (see Q2 Renewal path for documentation)
  3. Timeline: Renewal decision must be made BEFORE expiry date to avoid assessment gap
  4. Consequence: If renewal not processed by expiry, exemption ends and property becomes assessable again
```

Record-Keeping Checklist - What to Keep & How Long

Maintain complete records for every exemption decision. These form audit trail and support future review/renewal.

```
  • Decision Notice (KEEP 7 years): Written approval/rejection decision with reasons. File in correspondence folder.
  • Application Form & Documents (KEEP 7 years): Original submitted application and all supporting documents. Keep in order received.
  • Verification Log (KEEP 7 years): Document showing verification method, source, and result for each document. Essential for audit trail.
  • Title Register Copy (KEEP duration of exemption + 3 years): Dated title copy at time of approval. Proves ownership at that date.
  • Correspondence (KEEP 7 years): All emails, letters, documents exchanged with applicant, verifiers, other agencies.
  • Exemption Configuration (KEEP duration of exemption + 3 years): Snapshot of TaxNet exemption setup (screenshot of exemption fields, effective date, expiry date).
  • Escalation Documentation (KEEP 7 years): If case escalated, keep escalation request, supervisor notes, escalation decision.
  • Renewal Documents (KEEP each renewal cycle): For renewed exemptions, keep each renewal application and decision separately.

Storage Location

  • Physical Files: Land Revenue correspondence files, indexed by property address/V/F number
  • Electronic Files: Shared drive exemptions folder, organized by financial year and property
  • TaxNet History: TaxNet maintains audit trail of all exemption configuration changes - accessible via admin function
```

Exemption Withdrawal/Revocation Process (GAP 8 Addressed)

Describes how to withdraw or revoke an exemption that was previously granted.

```

When Exemption Must be Withdrawn

  • Property no longer meets exemption conditions (e.g., community use ceases, changed to private use)
  • Ownership transferred and new owner doesn't qualify
  • Exemption was granted in error (e.g., fraudulent documentation later discovered)
  • Legislative basis for exemption no longer applies
  • Exemption duration has expired and not renewed

Revocation Process

  1. Identify Need for Withdrawal: Review exemption status periodically. Identify if conditions still met.
  2. Notify Property Owner: Send written notice to registered proprietor (30 days advance notice if possible) explaining:
    • Reason for revocation
    • Effective date of revocation
    • Opportunity for owner to respond (14 days)
  3. Consider Response: If owner disputes revocation, escalate to manager for decision
  4. Issue Revocation Decision: Provide written final decision with reasons
  5. Update TaxNet: Remove exemption from TaxNet, set expiry date to revocation date
  6. Generate New Assessment: Property becomes assessable again. Issue updated assessment showing land tax now due.
  7. Record Documentation: Keep revocation notice, owner response (if any), and decision file with exemption records
```

Appeals Process - Applicant Disputes Decision

If applicant disagrees with exemption decision (approval or rejection), they have right to appeal.

```

Applicant Rights

  • Right to appeal decision within 60 days of decision notice
  • Right to have appeal reviewed by higher authority than original decision maker
  • Right to provide additional evidence or argument in appeal
  • Right to receive written decision on appeal with reasons

Escalation for Appeals

Note: Appeals process is documented in separate procedure "Appeals and Reviews - Exemption Decisions" procedure. Refer applicant to that procedure for steps and timelines.

Staff Role in Appeals

  • When receiving appeal, acknowledge in writing within 5 business days
  • Provide original decision file and all documentation to appeals reviewer
  • Do NOT make new decision on appeal - escalate to appropriate authority
  • Cooperate with appeals team to provide evidence or clarification if requested
```

Common Issues & Troubleshooting (GAP 10 Addressed)

```

Issue 1: Applicant Can't Provide Required Documents

Situation: Applicant has submitted application but is missing one or more required documents (e.g., statutory declaration, title certificate).

Solution: Issue written request for missing documents with 14-day deadline. Specify which documents needed and why. If deadline passes without response, reject application with notice of reasons for rejection.

Issue 2: Document Verification Fails

Situation: You attempt to verify supporting evidence (e.g., council approval letter) but the issuing body cannot confirm its authenticity.

Solution: Request applicant clarify or re-submit document from original source. If still unverifiable, escalate to manager. Do NOT approve exemption on weak/unverified documentation.

Issue 3: Legislative Reference Not Found

Situation: Applicant cites Act/Regulation section supporting exemption, but you search legislation and can't find that section or it doesn't say what applicant claims.

Solution: Escalate to Legal Services or Knowledge Management for legislative interpretation. Do NOT approve unless satisfied statutory basis exists.

Issue 4: Property Previously Rejected for Same Exemption

Situation: You find record that same property was previously rejected for exemption without material change in circumstances.

Solution: Reject immediately. Send written notice citing prior rejection decision and explaining no material change has occurred. Applicant can appeal if they disagree.

Issue 5: Exemption Expired - Customer Wants Retroactive Renewal

Situation: Exemption expired 6 months ago; customer only now applies for renewal.

Solution: Process renewal application immediately. If approved, backdated renewal possible only if property use continued unchanged and circumstances justify. Contact manager for guidance on backdating.

Issue 6: Conflicting Information in Application

Situation: Applicant's written statement about property use conflicts with supporting documentation or council records.

Solution: Request clarification in writing. If conflict unresolved, escalate to manager. Site inspection may be needed to verify actual current use.

```

Definitions and Acronyms

  • Other Exemption - Exemption from land tax that doesn't fit standard named categories; requires individual legislative assessment
  • Exemption - Freedom from land tax obligation on specified property, granted under legislative authority
  • Exemption Duration - Period for which exemption remains effective; may be indefinite or time-limited
  • Renewal - Process of extending exemption beyond initial expiry date; requires new application
  • Revocation - Cancellation of previously granted exemption; property becomes assessable again
  • Amendment - Change to existing exemption (e.g., ownership change, use change); less intensive than new application
  • Statutory Declaration - Formal sworn statement by applicant; must be signed before JP or lawyer
  • Legislative Authority - Act/Regulation/Order that permits exemption; must be cited in application
  • Land Tax - Annual tax levied on land valued over threshold; calculated on unimproved value
  • TaxNet - SRO's system for managing land tax assessments, exemptions, and billing
  • V/F - Volume/Folio number; unique identifier for registered land title
  • Landata - Public land title search tool; provides current title register information
  • Registered Proprietor - Legal owner of land as shown in Land Titles Office register
  • Crown Land - Land owned by State of Victoria; automatically exempt in most cases
  • Public Benefit - Use of property for benefit to public (not private); requirement for many exemptions
