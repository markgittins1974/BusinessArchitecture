Background: Valuation Objection Process CONTEXT

Overview of Valuation Objection Framework

Under Part III of the Valuation of Land Act 1960, customers may object to valuations on rate notices, ESVF notices, or land tax assessments (including VRLT). This procedure applies when a VCAT Order has been issued amending a valuation.

```

Objection Timeline

  • Customer objects: Within 2 months of assessment being issued
  • Valuer reviews: Has 4 months to recommend adjustment or disallow
  • VGV decides: Has 2 months to allow or disallow recommendation
  • VCAT review: Customer can appeal to VCAT if dissatisfied with valuer or VGV decision
  • VCAT Order: VCAT may amend valuation; SRO must comply
Legislation: Valuation of Land Act 1960 Part III, Taxation Administration Act 1997 Section 24A (interest on refunds)
Input data: VCAT Order, customer record, property valuation history
Processing responsibility: Staff (compliance and amendment), e-Sys (valuation recording)
Decision: Is order valid and ready to process?
Output: Amended valuation, recalculated liability, customer notification
```

Q1 - Has a Valid VCAT Order Amending a Valuation Been Received? IMPROVED

  • Order is issued by VCAT (not a recommendation or draft)
  • Order relates to land valuation (not other matters)
  • Order specifies amended valuation (original and new values shown)
  • Order identifies affected property/properties clearly
  • Order references are complete (VCAT case number present)
  • Order received from customer, valuer, or VGV (documented source)
```

Next Steps if Valid

  1. Save order in customer CM10 file (EDRMS)
  2. Record VCAT Reference on action item (Reference Tab)
  3. Proceed to Q2 (Action Item Check)
If order does NOT meet all 6 validity checks, escalate to Team Leader. Do NOT proceed with incomplete orders.
Procedure Detail: VCAT Order Validity Checklist▼

What Makes a VCAT Order Valid?

A valid VCAT Order must:

  • Be a Final Order (not interim, draft, or recommendation)
  • Be signed/issued by VCAT tribunal member
  • Have a VCAT case reference (e.g., VCAT Case 123/2025)
  • State the decision (e.g., "Application allowed" or "Valuation amended")
  • Specify original AND amended valuations
  • Identify property clearly (address, lot/plan, or property ID)
  • Specify year(s) to which amendment applies

NOT Valid - Do Not Process

  • VG Recommendation (not a VCAT Order - different process)
  • Draft or unsigned order
  • Order missing valuation amounts
  • Order relating to rates only (not land tax)
  • Photocopies without certification as true copy

If Order is Unclear

Contact VCAT or the customer for clarification. Do not proceed until order is confirmed as valid.

Source: Receiving VCAT Orders (LGA Procedure); Valuation of Land Act 1960 Part III
Input data: VCAT Order document, customer identification
Processing responsibility: Staff (validity assessment), EDRMS (document storage)
Decision: Is order valid (meets 6 criteria)?
Output: Valid → Save and proceed to Q2; Invalid → Escalate for clarification
```

Q2 - Does a Relevant Amendment Action Item Exist? IMPROVED

Check e-Sys for existing action items relating to this property and valuation level.

```
Which action item type exists?

• "Valuation Objection Amendment" (preferred for VCAT orders)
• "Other Amendment" (acceptable if already created)
• No action item (create new)
• Different type (change to "Valuation Objection Amendment" if possible)

If Action Item Exists (Same Property & Level)

  • Add notes to existing item with VCAT reference
  • Continue to Q3 (Apply Valuation)

If No Action Item Exists

  • Create new "Valuation Objection Amendment" action item
  • Record VCAT Reference (Reference Tab)
  • Proceed to Q3

If Different Type Exists

  • Contact Team Leader/Grade 3 to change type if possible
  • If change not possible, note reason and continue
Note: Special Customer handling - see Summary & Review view
Procedure Detail: Action Item Type Selection & Special Customer Handling▼

Action Item Type: Valuation Objection Amendment vs Other Amendment

Preferred: "Valuation Objection Amendment" - Used specifically for VCAT orders and valuation objection processes. Better tracking and easier to identify.

Acceptable: "Other Amendment" - Can be used if "Valuation Objection Amendment" not available in system.

Decision Rule: If only one type available, use that type. Record in comments that this is a VCAT order.

Special Customer Handling

If the customer is designated as a "Special Customer" in e-Sys (e.g., government agency, large corporation, disputed liability), you MUST:

  1. Discuss the VCAT order with assigned Senior Customer Service Officer BEFORE processing
  2. Document the discussion in action item comments
  3. Proceed only after Senior CSO approval

Special Customer Definition: Customers flagged in e-Sys as requiring special handling due to size, complexity, dispute history, or strategic importance.

Source: Receiving VCAT Orders (LGA Procedure); e-Sys Special Customer Classification
Input data: VCAT Order, e-Sys action item list for property
Processing responsibility: Staff (action item check), e-Sys (action item management)
Decision: Action item type - Valuation Objection vs Other vs Create new?
Output: Action item identified/created and recorded with VCAT reference
```

Q3 - Can the Amended Valuation Be Applied as Specified in the VCAT Order? IMPROVED

Step 1: Identify Affected Land

  1. Review VCAT Order to identify property/properties
  2. Confirm property matches customer's holdings
  3. If multiple lots, confirm all are listed on order
  4. Note current site value and capital improved value
```

Step 2: Check for Valuation Breakdown Requirement

Does VCAT Order provide single value for multiple lots?

• NO (single property) → Proceed to Step 3
• YES (multiple properties) → Request breakdown (see detail panel)

Step 3: Load Property to e-Sys

  • Load relevant property to e-Sys tree
  • Refer to VG Confirmation procedure for guidance

Step 4: Apply Amended Valuation

  • Valuation Type: "Objection"
  • Return Date: Unchanged (do not modify)
  • Site Value: Update per VCAT Order
  • Capital Improved Value: Update if VRLT affected
  • Comment: VCAT Reference + relevant notes
Do NOT apply valuation without required breakdown. Contact VGV for multi-property orders.
Procedure Detail: Valuation Breakdown, Multi-Property Orders, & Subdivision/Consolidation▼

When is Valuation Breakdown Required?

Breakdown is needed when:

  • VCAT Order provides single value for multiple properties
  • Common for shopping centres, multi-property estates
  • Breakdown must show: individual property value + total

How to Request Breakdown

Contact: Valuer General of Victoria or relevant valuation officer

Information to Provide: VCAT case reference, property details, original order

Timeline: Request within 5 working days of receiving order. VGV usually responds within 10-15 business days.

Multi-Property Handling Process

  1. Identify all properties in VCAT Order
  2. Load each property to e-Sys separately
  3. Apply breakdown values to each property
  4. Verify total matches VCAT order amount
  5. Document breakdown received in action item comments

Subdivision/Consolidation Scenarios

When subdivision may occur: If VCAT order addresses property that has been subdivided since original valuation.

When consolidation may occur: If VCAT order addresses consolidated property that was previously separate lots.

Action: If either situation suspected, escalate to Grade 3 for guidance on valuation application.

Valuation Type "Objection" System Field

This field identifies the valuation as resulting from a VCAT or objection process (not a revaluation cycle). Important for reporting and audit purposes.

Source: Processing a VCAT Order (LGA Procedure); VG Confirmation Procedure; e-Sys Valuation Fields Guide
Input data: VCAT Order (with valuations), property details, current valuation data
Processing responsibility: Staff (identification & application), e-Sys (valuation recording), VGV (breakdown if needed)
Decision: Can valuation be applied? (breakdown needed? subdivision/consolidation?)
Output: Amended valuation applied to relevant property/properties and years
```

Q4 - Reassess Liability, Calculate Refunds, and Notify Customer IMPROVED

Step 1: Recalculate Land Tax Liability

  1. Apply amended valuation to all relevant years
  2. Recalculate land tax liability for affected years
  3. Reissue assessment for each amended year
```

Step 2: Determine Overpayments & Refunds

Has amended valuation resulted in overpayment?

• NO overpayment → Proceed to Step 4
• YES overpayment → Calculate refund (with interest)

Step 3: Calculate Refund & Interest

  • Identify amount of overpayment
  • IMPORTANT: Interest applies under Section 24A TAA 1997
  • Interest accrues from original payment date to refund date
  • Refer to separate Land Tax Refunds procedure for calculation method
  • Arrange refund via EFT (electronic funds transfer)

Step 4: Prepare Customer Notification

  • Send email/letter to customer with:
  • VCAT Order reference
  • Table showing: property, year, original value, amended value
  • Notification of new assessment issuance date
  • Refund amount and payment method (EFT preferred)
  • Request for bank details if refund needed
Note: Email template provided in detail panel. Customization guidance included.

Step 5: Complete Action Item

  • Order saved to CM10 file
  • VCAT Reference recorded on action item
  • Valuation applied to all affected years
  • Liability recalculated
  • Refund calculated (if applicable)
  • Customer notification sent
  • All comments and notes documented
  • Action item closed
Procedure Detail: Refund Calculation, Interest Calculation, Customer Notification Template & Prior Year Amendments▼

Refund Determination Decision Tree

Question: Has amended valuation reduced land tax liability?

  • Yes: Customer overpaid. Refund is due.
  • No: Amended valuation is equal or higher. No refund.

Interest Calculation - Section 24A TAA 1997

Rule: Interest applies to refunds of overpaid land tax from objection/VCAT processes.

Calculation: Interest accrues from original payment date to refund date at current interest rate (set by Treasurer annually).

Application: For detailed interest calculation methodology, refer to Land Tax Refunds procedure.

Document Interest: Show interest separately in refund communication. Explain that interest is required by legislation.

Customer Notification: When & How

When to Send: After liability recalculated, assessment reissued, and refund calculated.

Method: Email preferred (faster); letter alternative if customer lacks email.

Customization: Use provided template as base. Customize only values and specific property details. Do NOT change legislative statements or refund conditions.

Prior Year Amendment Process

Situation: VCAT Order amends valuation for years not in current assessment cycle.

Process:

  1. Identify all years affected by VCAT Order
  2. Load each year's assessment to e-Sys
  3. Apply amended valuation per order
  4. Recalculate liability for each year
  5. Reissue assessment for each amended year
  6. Calculate cumulative refund if multiple years affected

Documentation: Record all years amended in action item comments and customer communication.

Source: Processing a VCAT Order (LGA Procedure); Taxation Administration Act 1997 Section 24A; Land Tax Refunds Procedure
Input data: Amended valuation (applied), recalculated liability, current/prior year assessments
Processing responsibility: e-Sys (liability recalculation), Staff (refund determination), Refunds team (refund processing)
Decision: Overpayment? Refund amount? Prior years affected?
Output: Reissued assessments, refund initiated, customer notified, action item completed
```

VCAT Orders vs VG Confirmation: Key Distinctions

Understanding the difference is critical for applying the correct procedure.

```
Aspect VCAT Order VG Confirmation
Origin VCAT tribunal (judicial body) Valuer General of Victoria recommendation
Authority Binding final decision (appealable only to higher courts) Recommendation from VGV assessment process
Procedure to Use THIS procedure (VCAT Orders) VG Confirmation procedure (different)
When Occurs After customer appeals VGV decision to VCAT After valuer completes assessment review
Action Item Type Valuation Objection Amendment Valuation Objection Amendment (same)
Refund Possible Yes (if overpaid) Yes (if overpaid)
```

Special Customer Handling Process

What is a Special Customer? Customers designated in e-Sys as requiring special handling due to: government agency status, large corporation, dispute history, strategic importance, or complexity.

```

Escalation Process for Special Customers

  1. Identify customer as "Special" in e-Sys
  2. BEFORE processing VCAT Order: Discuss with assigned Senior Customer Service Officer
  3. Document discussion in action item comments
  4. Obtain Senior CSO approval before proceeding
  5. Proceed with normal processing steps
Never process a VCAT Order for a Special Customer without Senior CSO notification and approval.
```
